Australian Open Tournament Statistics

What is your chance to win the match?

Assignment 2 Write-Up


Data Analysis:
In the given data, there are 37 columns and 1387 matches data. Every match data has the statistic of winner and loser. For data analysis and data cleaning part, python is used. Various libraries of python like Numpy, Pandas, matplotlib and seaborn are used for the data analysis and feature selection.

As every match has winner and loser statistics, those can be used to determine the most important features to win a match. Using matplotlib and seaboarn, co relation between every feature was obtained. In addition to that, linear regression was also done on the various combination of features. By analyzing the result, eight most important features were selected.

Columns which are selected as most important features are:
ACE     RETURN     BREAK     DOUBLE     ERROR     FIRSTPOINTWON     SECONDPOINTWON     TOTAL
Among these columns        RETURN, BREAK, EROOR     have missing data.
As the probability for winning the match with every feature is to be calculated, if missing value is replaced with the mean value than also it will not affect the result much. So Missing values are replaced with mean values. With the use of linear regression, weight of each feature to contribute in the winning is decided.

Now, for each feature, frequency of the feature in a specific range is calculated. Range is decided using maximum and minimum value of the feature and by dividing with the total number of matches will give the probability for each range. So, if FirstPointWon feature has maximum possible value is 100% and minimum possible value is 0% than frequency for the ranges 0-20, 20-40, 40-60, 60-80, 80-100 are calculated and dividing that with the total number of matches will give probability for each range. Same way, the all the probabilities for all the features are calculated.

Story:
In any tennis match, there are so many factors like ace, break, return, etc. which affects the result of the tennis match. But the exact effect of each feature and effect on winning for a given feature value is very difficult task.
With this story, above issue is addressed and winning probability is calculated with taking input from the user for each feature.

Interactive Visualizations
For taking input, slider is implemented as it is very intuitive and easily draggable.
For showing analysis of probability for each feature with weight, aster graph is used.
Aster graph is basically pie chart with different weight to every feature so it can easily represent the feature weight and probability of that feature.
